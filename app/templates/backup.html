Agendamento de Backups Automáticos

Para automatizar a criação de backups, você pode usar o agendador de tarefas do seu sistema operacional para executar o script backup_scheduler.py.

Exemplo para Windows (via Agendador de Tarefas):
  1. Abra o "Agendador de Tarefas".
  2. Crie uma nova tarefa e defina um gatilho (ex: diariamente às 02:00).
  3. Na aba "Ações", crie uma nova ação com as seguintes configurações:
    • Ação: Iniciar um programa
    • Programa/script: C:\caminho\para\seu\python.exe (o caminho para o executável do Python no seu ambiente virtual .venv\Scripts\python.exe)
    • Adicione argumentos (opcional): C:\caminho\completo\para\o\projeto\backup_scheduler.py
Exemplo para Linux/macOS (via Cron):
  1. Abra o terminal e digite crontab -e.
  2. Adicione a seguinte linha para executar o backup todos os dias às 2 da manhã:
0 2 * * * /caminho/para/seu/python /caminho/completo/para/o/projeto/backup_scheduler.py
Lembre-se de substituir os caminhos pelos diretórios corretos do seu ambiente e projeto.
